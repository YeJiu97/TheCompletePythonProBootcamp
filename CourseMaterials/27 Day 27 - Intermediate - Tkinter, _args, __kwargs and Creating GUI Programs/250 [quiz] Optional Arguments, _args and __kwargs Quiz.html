Question 1

What is the output of this code?


def bar(spam, eggs, toast='yes please!', ham=0):
    print(spam, eggs, toast, ham)

bar(1, 2)


1
2
3
4

Correct Answer

1

1 2 'yes please!' 0

Explanation

The optional arguments, toast and ham, get assigned their default values.

Question 2

What is the output of this code?


def bar(spam, eggs, toast='yes please!', ham=0):
    print(spam, eggs, toast, ham)

bar(toast='nah', spam=4, eggs=2)

1
2
3
4

Correct Answer

3

4 2 'nah' 0

Explanation

Correct. When keywords are used the order of the arguments listed does not matter. Python matches by name not position.

Question 5

def all_aboard(a, *args, **kw): 
    print(a, args, kw)

all_aboard(4, 7, 3, 0, x=10, y=64)


What is the output of the code above? 

1
2
3
4
5

Correct Answer

3

4 (7, 3, 0) {'x': 10, 'y': 64}

Explanation

Correct. 4 is passed by position. 7,3,0 are collected into a tuple. x and y are in a keyword dictionary
