1. Download the starting .zip files from this lesson's resources (Starting Files - cafe-api-start.zip).


2. Unzip and open the project in PyCharm.


3. Make sure that the required packages (imports) are all installed and there are no red underlines.


The starting files consists of an SQLite database called cafes.db, I created this in the same way we created databases before. I added a bunch of records consisting of cafes around London. These are some of my favourite places to work from.

4. Take a look a the database using DB Viewer and familiarise yourself with the fields in the database.
