Instructions

Let's review some of the concepts around class inheritance.

Question 1

Given the following:

class Dog:
    def __init__(self):
        self.temperament = "loyal"

    def bark(self):
        print("Woof, woof!")

How do you create a class called Labrador (the subclass) that inherits from the Dog class (the superclass)?

1
2
3
4

Correct Answer

2 
class Labrador(Dog):
    def __init__(self):
        self.temperament = "friendly"

Explanation

Correct. The call to super() in the initialiser is recommended, but not strictly required.

Question 2

class Dog:

class Labrador(Dog):

What will this code print?

doggo = Dog()
print(f"A dog is {doggo.temperament}")

sparky = Labrador()
print(f"Sparky is {sparky.temperament}")

Question 3

Given this code:

class Dog:
    def __init__(self):
        self.temperament = "loyal"

    def bark(self):
        print("Woof, woof!")

class Labrador(Dog):
    def __init__(self):
        super().__init__()
        self.is_a_good_boy = True

    def bark(self):
        super().bark()
        print("Greetings, good sir. How do you do?")


What will this print?

sparky = Labrador()
sparky.bark()

1
2
3
4

Correct Answer

3

Woof, woof!

Greetings, good sir. How do you do?

Explanation

Correct. We are extending the functionality of the bark() method.
